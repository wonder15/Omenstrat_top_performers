# Omenstrat_top_performers
These scripts find all addresses that were active in OMEN markets the last 30 DAYS and returns a table that tabulates a selection of high performing agents. Optionally, depending the chosen script, 2-week-to-date ROI, month-to-date ROI, or lifetime ROI are returned. 

INPUT: API key for the Graph the Omen subgraph, and the mech subgraph as endpoints. 

OUTPUT: A table of top performing agents.
